# PyCalculator

The application is a simple tool for performing basic arithmetic operations. 

The user has 4 operations to choose. 
```
MENU:
1 - Addition 
2 - Subtraction
3 - Multiplication 
4 - Division
```

3 tool forms are available:
- REST API from http://localhost:5000
- From the terminal, built on:
  * class
  * loop if
